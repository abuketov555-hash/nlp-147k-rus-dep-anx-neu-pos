# nlp-147k-rus-dep-anx-neu-pos

Проект посвящён задаче классификации русскоязычных текстов по четырём классам тональности:

- `dep` — depressive
- `anx` — anxious
- `neu` — neutral
- `pos` — positive

## Описание корпуса

Обучающий корпус построен из 146811 текстов, разделенными по 4 классам: pos: 48965, dep: 38995, neu: 37678, anx: 21173. Использованы открытые русскоязычные данные c разметкой: корпус «Taiga», корпус «multilingual-mental-health-datasets-nlp», база данных «RusIdiolect», корпус «LLM_psycho_mark_up_», набор данных «Dataset of depressive and suicidal posts», а также корпус «pikabu». 

## Репозиторий

В репозитории собран код для экспериментов по векторизации текста и классификации представленного корпуса.  
Рассматриваются классические и нейросетевые подходы к представлению текста, а также их гибридные комбинации в одноуровневом и двухуровневом исполнении.

Основные направления экспериментов:

- TF-IDF
- Word2Vec
- GloVe
- FastText
- SBERT
- гибридные модели семейства TFIDF-SBERT

## Цель проекта

Цель проекта — сравнить различные подходы к векторизации и классификации текста, построить модель классификации для задачи распознавания психоэмоционального состояния по письменной речи.
