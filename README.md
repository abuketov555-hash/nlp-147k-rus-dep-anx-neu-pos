# nlp-147k-rus-dep-anx-neu-pos

Проект посвящён задаче классификации русскоязычных текстов по четырём классам тональности:

- `dep` — депрессивный
- `anx` — тревожный
- `neu` — нейтральный
- `pos` — позитивный

## Описание корпуса

Обучающий корпус построен из 146811 текстов, разделенными по 4 классам — pos: 48965, dep: 38995, neu: 37678, anx: 21173. Использованы открытые русскоязычные данные c разметкой: корпус «Taiga», корпус «multilingual-mental-health-datasets-nlp», база данных «RusIdiolect», корпус «LLM_psycho_mark_up_», набор данных «Dataset of depressive and suicidal posts», корпус «pikabu». 

## Тестовый корпус

Для оценки устойчивости моделей к смещению распределения был также построен тестовый корпус на 1593 примера, разделенных по 4 классам как pos: 386, dep: 417, neu: 393, anx: 397. Корпус имеет иное распределение, чем обучающий (см рисунки). 

## Репозиторий

В репозитории собран код экспериментов по векторизации текста и классификации представленного корпуса.  
Рассматриваются классические и нейросетевые подходы к представлению текста, а также их гибридные комбинации в одноуровневом и двухуровневом исполнении.

Основные направления:

- TF-IDF + LinearSVC
- Word2Vec + LinearSVC
- GloVe + LinearSVC
- FastText + LinearSVC
- SBERT(-encoder) + LinearSVC
- гибридные модели вида TFIDF-SBERT + LinearSVC/LogReg

## Цель проекта

Цель проекта — сравнить различные подходы к векторизации и классификации текста; построить модели классификации для задачи распознавания психоэмоционального состояния по письменной речи на базе классических моделей и LLM.

## Результаты проекта

Построенные модели позволяют достичь точности классификации по метрике Accuracy: 0.745-0.945 для базововго корпуса и  0.401-0.655 для тестового корпуса на базовых моделях и 0.954-0.967 для базового корпуса и 0.680-0.711 для тестового корпуса на гибридных моделях.
